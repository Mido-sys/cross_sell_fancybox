INSTALLATION

  2. Install CSS JS Loader module
  3. Upload all files in this package while maintaining the directory structure (rename YOUR_TEMPLATE to your custom template folder name and YOUR_ADMIN to your custom admin folder name).
  4. Open includes/templates/YOUR_TEMPLATE/template/tpl_modules_main_product_image.php and find:

UNINSTALL
  1. Backup files and database.
  2. Remove edits performed in the installation procedure above.
  3. Delete all uploaded files from your store.
  4. Patch your databasein ADMIN > TOOLS > INSTALL SQL PATCHES with the included uninstall.sql file located in the root of the installation package .
TIPS
  • The default JavaScript pop-up for product images conflicts with this mod. If you are using a lightbox plugin for your product images, you do not need to perform the edit to the tpl_modules_product_main_image.php as mentioned above.
  • If your template does not have an ideal container for the content you'd like to load in the fancybox, you can make your own div surrounding the content. Just be sure to specify the id of that div in the configuration. Note: Ensure the container your select or create contains the opening and closing form elements or the add to cart will not work.
